# http_serve.py

## English

A simple HTTP server implementation in Python that serves files from the current directory.

### Usage

Run the script:
```bash
python http_serve.py
```

The server will start on port 8000 and serve files from the current directory.

### Features

- Simple file server
- Runs on port 8000
- Serves files from the current directory
